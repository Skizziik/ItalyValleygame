# Italy Valley -- Asset Sources and Licenses

> Track every external asset used in the project. Keep this file up to date whenever a new asset is added.

---

## How to Use This File

1. Before adding any asset (sprite, sound, font, music, shader, etc.), record it here.
2. Fill in every column. If unsure about a license, research it before using the asset.
3. Assets with "Yes" in the Attribution Required column MUST have visible credit in the game's credits screen and/or README.
4. Never use assets with licenses incompatible with the project's distribution model.
